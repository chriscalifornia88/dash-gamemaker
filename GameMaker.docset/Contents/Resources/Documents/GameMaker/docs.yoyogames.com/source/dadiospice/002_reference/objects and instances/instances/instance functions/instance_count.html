instance_count

Returns the total number of instances currently active in the room.

Syntax:

instance_count


Returns: Real


Description

With this read only variable you can get a count of all active instances that are in the room. This will include the instance running the code, but does not include those instances that have been deactivated using the instance_deactivate functions.


Example:

while (instance_count < 100)
   {
   instance_create(random(room_width), random(room_height), obj_Star);
   }

The above code will create multiple instances of the object "obj_Star" until the total instance count reaches 100.
