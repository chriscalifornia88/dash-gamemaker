steam_available_languages

Returns the available languages for Steam.

Syntax:

steam_available_languages();


Returns: String


Description

This function can be used to retrieve a list of all available languages for Steam. The returned value is simply a a comma separated list of languages.


Example:

language = steam_available_languages();

The above gets the available languages for Steam as a string and stores it in a variable.
